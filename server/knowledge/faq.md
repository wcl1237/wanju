# 常见问题 (FAQ)

## 账户相关

### 如何注册账号？
访问我们的官方网站，点击右上角的"注册"按钮。填写您的邮箱、手机号和密码即可完成注册。注册后需要通过邮箱或短信验证码进行身份验证。

### 忘记密码怎么办？
点击登录页面的"忘记密码"链接，输入您注册时使用的邮箱或手机号，系统将发送重置密码的链接或验证码。按照提示操作即可重置密码。

### 如何修改个人信息？
登录后进入"个人中心" > "账户设置"，您可以修改昵称、头像、联系方式等个人信息。修改手机号和邮箱需要进行安全验证。

## 订单相关

### 如何查看订单状态？
登录后进入"我的订单"页面，可以查看所有订单的当前状态。订单状态包括：待付款、已付款、处理中、已发货、已完成、已取消。

### 如何取消订单？
在"我的订单"中找到需要取消的订单，点击"取消订单"按钮。注意：已发货的订单无法直接取消，需要联系客服处理。订单取消后，退款将在1-3个工作日内原路退回。

### 退款需要多长时间？
退款时间取决于支付方式：
- 支付宝/微信支付：1-3个工作日
- 银行卡：3-7个工作日
- 信用卡：7-15个工作日

## 技术支持

### 支持哪些浏览器？
我们的服务支持以下主流浏览器：
- Chrome 80+
- Firefox 78+
- Safari 14+
- Edge 80+
建议使用最新版本的浏览器以获得最佳体验。

### API 调用频率限制是多少？
免费用户：每分钟 60 次请求
标准用户：每分钟 300 次请求
企业用户：每分钟 1000 次请求
如需更高限额，请联系商务团队。

### 数据安全如何保障？
我们采用业界领先的安全措施：
- 数据传输：全程 HTTPS/TLS 加密
- 数据存储：AES-256 加密
- 访问控制：RBAC 权限模型
- 定期进行安全审计和渗透测试
